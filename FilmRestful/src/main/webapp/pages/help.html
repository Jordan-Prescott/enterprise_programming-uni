Help Page

This page provides detailed information about this web application. Details on all features and step by step instructions on how to achieve a task can all be found in here. The structure of this help page is page first and then each feature beneath for example if you wanted to look for details on the search function you would first locate Main Page as this is the page its located on and then look for search.

There is a table of content you can use to click to the relevant page you are looking for.

Main Page

The main page is where all films in the database are presented in a table and where you access all features of this application. This is a simple page with a navigation bar across the top and the films table in the body.

Navigation Bar

The navigation is to help you navigate around the application as well as hosting a couple other features such as notifications and search. This bar contains a home button of 'Film DB' a link to this help page, notifications section and the search function on the right-hand side.
